# Voter Info Web

A web app for searching and browsing voter records from Dharmpur, Sitakunda, Chittagong. Supports Bengali text.

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm run start
```

## Tech Stack

Next.js 15, TypeScript, Tailwind CSS, shadcn/ui, Fuse.js, TanStack Table, Zustand

## Data

Place voter data in `public/voter_data.json`. Contains 2,305 records (1,253 male, 1,052 female).
